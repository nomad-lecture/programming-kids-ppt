# Programming Kids PPT

5-р ангийн хүүхдүүдэд зориулсан програмчлалын тухай PPT үүсгэх script.

## Install
```bash
pip install -r requirements.txt
```

## Run
```bash
python generate_ppt_with_images.py
```
